### Description

This loading screen was used by all the talents as they prepared to begin their day in Chapter 2. It features a looping animation of a spinning tarot card, with each rotation revealing a different hero, set against a starry, mystical backdrop.

The background music is the instrumental version of **MONSTER**, the official theme song of Chapter 2.
